# 房地合一稅自住優惠與重購退稅檢核系統

這是一個基於 Web 的單頁式工具，旨在協助使用者快速檢核其房屋土地交易是否符合「房地合一稅 400 萬自住免稅優惠」與「重購退（抵）稅」之法定要件。

## 🚀 系統功能

- **民國曆下拉選單**：全面採用民國年選擇介面，避免西元曆換算誤差。
- **自住要件自動檢核**：
    - **400 萬免稅額**：檢查持有並設籍是否連續滿 6 年、無出租營業、6 年內未曾適用。
    - **重購退（抵）稅**：計算新舊屋移轉間距（2 年內）、產權關係、設籍對象等。
- **5 年列管期警示**：包含 **財政部 111 年 6 月 9 日台財稅字第 11100584750 號令** 之例外情形判定（如子女就學、公務派駐等）。
- **視覺化儀表板**：
    - **事件時間軸**：依時序排列取得、設籍、移轉日期。
    - **動態甘特圖**：直觀顯示持有與設籍時間長短，與時間軸並列對照。

## 🛡️ 安全性與隱私 (Security & Privacy)

本專案在設計上極度重視使用者隱私與資料安全：
1. **無 API 介接**：本程式不使用任何外部 API Key（如 Google API），亦不涉及政府資料介接。
2. **100% 本地運算**：所有稅務試算與日期邏輯皆在您的瀏覽器端執行（Client-side）。
3. **資料不外洩**：使用者輸入的所有成交價格、日期等敏感資料**不會傳送到任何伺服器或後端資料庫**。
4. **透明程式碼**：純靜態 HTML/JavaScript 結構，程式碼公開透明，無任何惡意腳本或追蹤程式。

## 🛠️ 技術架構

- **前端框架**：React 18 (透過 CDN 載入)
- **樣式工具**：Tailwind CSS
- **編譯器**：Babel Standalone
- **佈署平台**：建議使用 GitHub Pages

## 📦 佈署教學

1. 將本專案的 `index.html` 上傳至 GitHub 儲存庫。
2. 進入該儲存庫的 **Settings** > **Pages**。
3. 在 **Build and deployment** > **Branch** 選擇 `main` (或 `master`) 分支。
4. 點擊 **Save**。
5. 約一分鐘後，即可透過 GitHub 提供之網址線上執行此工具。
